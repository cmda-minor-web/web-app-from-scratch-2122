Opdracht Null Kickoff Minor Web Design & Development

Opdrachtomschrijving

In de minor gaan we werken in kleine teams. Drie teams samen vormen een Squad (fka klas) en de twee Squads samen vormen de Tribe, ofwel de hele groep. Om elkaar beter te leren kennen, maakt iedereen een eigen digitaal visitekaartje . Daarna maken jullie samen per team een team-pagina waarin jullie je presenteren aan de Tribe

Teamindeling

Opdracht 0.1: Maak een digitaal visitekaartje

  • Fork repo en zet GitHub pages aan
  • Haal jouw data op uit de Teams API
  • Render data in HTML en geef het vorm met CSS
  • Update de lege velden van jezelf in de API: mugshot, github-handle en eventueel other
  • Update de code op GitHub / GitHub Pages

Bronnen

Documentatie Minor Web API (MOCK API) Haal data op uit een API (MDN)

Opdracht 0.2: Maak een webpage voor het team

  • Bedenk een teamnaam & ontwerp een logo (optioneel)
  • Wijs een team leader aan
  • Maak een repo aan voor het team.
  • Haal data van alle members van het team op uit de Teams API
  • Render data in HTML en geef het vorm met CSS (bedenk samen een huisstijl)
  • Update de code op GitHub / GitHub Pages
